Mac 快捷键

Command + ~ 切换同一应用的多个窗口
Commond + W 关闭当前窗口
Commond + N 新建窗口

Commond + Shift + 4 自由截图
Commond + Shift + 4 + 空格键 截取当前窗口

Commond + Left/Right 行首行尾
Commond + Up/Down 页首页尾
Option + Left／Right 按单词移动
